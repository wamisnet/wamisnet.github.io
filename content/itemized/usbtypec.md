# USB Type C変換基板

## なにができるの？
USB Type CのソケットからUSB2.0で使われるピンをブレットボードなどで使いやすい2.54mmピッチに変換する基板です。

ピンソケットをつけない場合  
![nopin](/img/item/usbnopin.jpg)  
ピンソケットをつけた場合  
![pin](/img/item/usbpin.jpg)
